# ShopZone — E-Commerce Store

A fully responsive E-Commerce store built with React and Tailwind CSS. Features complete shopping experience with cart management, wishlist, blogs, and checkout flow.

## 🔗 Live Demo
[View Live Project](https://ecommerce-store-mursal.vercel.app)

## ✨ Features

- 🏠 Home Page — Hero banner, categories, featured products, sale banner
- 🛍️ Products — Grid layout, search, filter by category, sort by price/rating
- 📦 Product Detail — Images, ratings, quantity selector, add to cart/wishlist
- 🛒 Cart — Add/remove items, quantity update, order summary, free delivery threshold
- ❤️ Wishlist — Save products, move to cart
- 📝 Blogs — Blog listing, search, filter by category, full blog detail page
- 🔐 Auth — Login, Register with validation
- 💳 Checkout — Shipping form, payment method (Card/COD), order confirmation
- 🌙 Dark / Light Mode — Full theme toggle
- 📱 Fully Responsive — Mobile + Tablet + Desktop

## 🛠️ Tech Stack

- React 18
- Tailwind CSS
- Context API (Cart & Wishlist)
- Lucide React
- Vite
